How this works
Status badges — how the file was matched to an existing chart:
  • Exact Same Drive file ID, file has changed — will replace the existing PDF.
  • High No stored file ID but one chart matches this instrument/part — likely a re-export.
  • Needs review Multiple files map to the same instrument/part — pick the right one.
  • New No existing chart found — will create a new one.
Instrument confidence icons — how certain the parser is:
  • ✓ High confidence — instrument name matched exactly or via known alias.
  • ⚠ Ambiguous — name could refer to more than one instrument; choose from the dropdown.
  • ? Low confidence — no strong match found; review and correct before importing.
Part field — leave blank for single-part charts. Use 1st Trumpet, 2nd Trumpet etc. for multi-part charts.
Action column — new creates a chart; update replaces the PDF on an existing chart and deletes the old file. Uncheck any row to skip it.
