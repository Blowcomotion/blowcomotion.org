Import Charts from Google Drive

Each row is a Google Drive folder. Click Review to scan its PDF files and import charts for that song.

The Auto-detected song column shows which song in the database was matched to the folder name. If it looks wrong, you can correct it on the review page before importing. A low match confidence means the name didn't match closely — double-check before proceeding.

File naming guide

The importer reads the instrument and part from each PDF filename. Supported formats:

  • Song – Instrument.pdf — e.g. Caravan - Trumpet.pdf
  • Song – Instrument 2nd.pdf — e.g. Caravan - Trombone 2nd.pdf or Caravan - Clarinet in Bb1.pdf
  • Song-Instrument.pdf (no spaces around dash) — post-dash side is the instrument
  • Song – Bb.pdf / Song – Eb.pdf / Song – C.pdf — transposition key: one PDF shared across multiple instruments (Bb = Trumpet/Clarinet/Tenor Sax, Eb = Alto/Bari Sax, C = Flute/Trombone/Tuba)
  • Song Score.pdf / Song Full Score.pdf / Song Conductor.pdf / Song All Parts.pdf — treated as conductor/score, not linked to an instrument

Common abbreviations are recognized automatically: tbn, tmpt, bari sax, alto sax, tenor sax, euph, sousa, etc. Part numbers work as digits, ordinals, or roman numerals (1, 1st, I).
