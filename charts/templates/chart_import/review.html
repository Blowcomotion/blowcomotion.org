Import Charts / Review

Review Charts Status badges — how the file was matched to an existing chart:
  • Exact Same Drive file ID, file has changed — will replace the existing PDF.
  • High No stored file ID but one chart matches this instrument/part — likely a re-export.
  • Needs review Multiple files map to the same instrument/part — pick the right one.
  • New No existing chart found — will create a new one.
Instrument confidence icons — how certain the parser is:
  • ✓ High confidence — matched exactly or via known alias.
  • ⚠ Ambiguous — choose from the dropdown.
  • ? Low confidence — review and correct before importing.
  • Conductor Score — full score/conductor chart. Uncheck the checkbox to assign a specific instrument instead.
Part field — leave blank for single-part charts. Use 1st Trumpet, 2nd Trumpet etc. for multi-part.

Action column — new creates a chart record; update replaces the Drive URL on an existing one. Uncheck any row to skip it.

Multi button — switch a row to a multi-instrument selector when one PDF covers multiple instruments. Each checked instrument gets its own chart record.
